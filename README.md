# Quran Ayah Generator

A simple, peaceful web app that displays random Ayahs from the Quran. Built with HTML, CSS, and Vanilla JS.

## How it Works

The Quran contains 6,236 Ayahs. The app generates a random number between 1 and 6,236 and uses it to fetch a specific Ayah from the [Al Quran Cloud API](https://alquran.cloud/api).

*** You can deploy it in the browser or access using github pages [here](https://darealsalim01.github.io/ayah-generator/)
